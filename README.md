# Jmetav

A data scraper for Jav written in TypeScript with Bun.

## Compile and Run

```sh
git clone https://github.com/glyh/jmetav.git
cd jemtav
bun install
bin/cli # See bin/cli --help to have all available params.
```

## Installing

```sh
git clone https://github.com/glyh/jmetav.git
cd jemtav
bun install
bun link
```
